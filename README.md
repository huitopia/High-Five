    <h2> "경매"를 통해 중고 상품을 거래하는 플랫폼
    </h2>
    <p>
        타임어택⏳ 경매로 경매의 스릴과 긴장감을 극대화하는 새로운 방식입니다.<br />

         당첨된 후에는 판매자와 바로 채팅을 통해 구매를 확정할 수 있어, 빠르고 편리한 거래를 경험할 수 있습니다.
